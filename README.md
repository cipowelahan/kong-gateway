# Kong API Gateway

need docker & docker-compose

## How to use ?

just run this command to deploy up `(maybe :V)`:

```bash
sh deploy.sh
```

### Option

`--down` to down all service

`--no-konga` to not create konga (GUI Admin)

`example`:

```bash
sh deploy.sh --down
```
